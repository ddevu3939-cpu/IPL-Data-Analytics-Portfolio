# IPL-Data-Analytics-Portfolio
# IPL Data Analytics Portfolio Project

## Project Overview

This project demonstrates an end-to-end Data Analytics workflow using the Indian Premier League (IPL) dataset. The project includes data cleaning with Python, business analysis using SQL, interactive dashboard creation in Power BI, and project documentation on GitHub.

---

## Tools Used

- Python
- Pandas
- SQL (SQLite)
- Power BI Desktop
- GitHub

---

## Project Workflow

### 1. Data Cleaning (Python)

- Loaded the IPL dataset
- Checked data types
- Converted the date column into datetime format
- Handled missing values
- Saved the cleaned dataset as `IPL_Cleaned.csv`

---

### 2. SQL Analysis

Performed business analysis using SQL queries, including:

- Total Matches
- Total Runs by Team
- Top 10 Batters
- Top 10 Bowlers
- Matches by Season
- Player of the Match Awards
- Toss Decision Analysis
- Top Venues by Matches

SQL queries are available in `IPL_Analysis.sql`.

---

### 3. Power BI Dashboard

Created an interactive dashboard containing:

#### KPI Cards

- Total Matches
- Total Runs
- Total Wickets
- Total Teams

#### Charts

- Runs by Team
- Matches by Season
- Top 10 Batters
- Top 10 Bowlers
- Toss Decision Analysis
- Top Venues

#### Slicers

- Season
- Batting Team
- Bowling Team
- Venue
